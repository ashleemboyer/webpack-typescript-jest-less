# A React App with webpack 5, TypeScript, Less Modules, and Jest

## To install dependencies

```bash
yarn
```

## To run locally

```bash
yarn dev
```

## To run tests

```bash
yarn test
```

## To build

```bash
yarn build:dev
```

or

```bash
yarn build:prod
```
